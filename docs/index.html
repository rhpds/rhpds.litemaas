What is RHDP MaaS?

RHDP MaaS is the Red Hat Demo Platform's Model-as-a-Service implementation — giving users managed access to AI/LLM models with API key management, subscriptions, usage tracking, rate limiting, and budget controls. Users get a simple OpenAI-compatible endpoint without managing any infrastructure.

RHDP MaaS is built on LiteMaaS, an open-source platform by the Red Hat AI Services BU. This documentation covers the RHDP-specific deployment using the rhpds.litemaas Ansible collection.
