What is RHDP LiteMaaS?

LiteMaaS is a Model-as-a-Service platform built on LiteLLM that provides Red Hat Demo Platform users with managed access to AI/LLM models. It handles API key management, model subscriptions, usage tracking, rate limiting, and budget controls — so workshop attendees and developers get a simple API endpoint without managing infrastructure.
